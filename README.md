# EcoTrack 🌱

EcoTrack is a web application that helps individuals understand, track, and reduce their carbon footprint through personalized insights, sustainability challenges, and progress tracking.

## 🚀 Live Demo

https://ecotrack-carbon-foot-tobj.bolt.host

## ✨ Features

* Carbon footprint calculator
* Personalized sustainability recommendations
* Progress dashboard and analytics
* Daily eco challenges
* Goal tracking system
* Community leaderboard
* Carbon offset exploration
* Secure authentication with Supabase

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Supabase

## 📊 Categories Tracked

* Transportation
* Electricity Usage
* Food Consumption
* Waste Generation

## 🎯 Objective

EcoTrack aims to make climate action measurable and accessible by helping users understand the environmental impact of their daily habits and encouraging sustainable behavior through data-driven insights.

## ⚙️ Installation

```bash
git clone https://github.com/hashhhh-16/ecotrack.git
cd ecotrack
npm install
npm run dev
```
